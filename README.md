# COMS Steering

## Description

Package for controlling the steering wheel for the COMS EV project.

## Dependencies

- serial

## Building and Running

```
$ cd ~/ros/catkin_ws/src
$ git clone https://github.com/naoki-mizuno/coms_steering
$ cd ~/ros/catkin_ws
$ catkin_make
$ rosrun coms_steering coms_steering_node
```
